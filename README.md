# Cocoa Programming Exercises

Practical implementations of exercises and challenges from the book **"Cocoa® Programming for Mac® OS X, Fourth Edition"** by Aaron Hillegass and Adam Preble.

## Overview
This repository contains my hands-on solutions while learning Objective-C and Cocoa framework for macOS development. Each exercise is a separate Xcode project, built, tested, and experimented with as part of the learning process.

## Structure
- Each directory corresponds to a specific chapter or exercise from the book.
- Projects are named according to the exercise topic.
- All code is manually typed, not copied, to reinforce understanding.
